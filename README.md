# En desarrollo

1. Clonar el repo.
2. Ejecutar el comando de:

```bash
$ yarn install
```

# Para levantar el servidor

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
